O que são Tags Semânticas?

As tags semânticas são elementos HTML que têm um significado específico e ajudam a estruturar o conteúdo de forma mais clara e acessível.

Exemplos de Tags Semânticas

Header

A tag <header> é usada para definir um cabeçalho para um documento ou seção.

Footer

A tag <footer> é usada para definir um rodapé para um documento ou seção.

Article

A tag <article> é usada para definir um conteúdo independente que pode ser distribuído ou reutilizado.

Section

A tag <section> é usada para definir seções em um documento, agrupando conteúdo relacionado.
